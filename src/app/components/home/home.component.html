Bienvenido al Sistema

INSTRUCCIONES
  1. Seleccione una opción del menú lateral para comenzar.
  2. Diríjase a Registrar Pacientes para ingresar los datos del cliente.
  3. Luego acceda a Agendar Citas para seleccionar una fecha y horario.
  4. Elija un paciente y haga clic en un horario disponible (color verde).
  5. Los horarios reservados aparecerán en color rojo.
  6. Puede gestionar las citas seleccionando diferentes pacientes.
  7. Revise constantemente la agenda para evitar duplicaciones.

NOTA: Para un correcto funcionamiento, registre primero al paciente antes de agendar una cita.
